Printer Simulator Test Printers

The Printer Simulator provides five virtual test printers that report different capabilities.

Save Original to Simulator

Choosing this printer will save the original PDF or image file to the location specified in your preferences. Print options including page range and copies will be ignored.

Simulated 2-Sided Inkjet

This printer simulates a color raster printer that supports 2-sided printing. Print images in Preview will show the back side images (even numbered pages) upside-down when 2-sided printing is enabled.

Simulated Color Laser

This printer simulates a color PDF capable printer that supports 2-sided printing.

This printer simulates media-size sensing capabilities. The printer informs the iOS printing system that it has US Letter paper loaded in the main tray. iOS will therefore choose this paper size regardless of what region the user has set.

Simulated Inkjet

This printer simulates a color raster printer that supports single-sided printing only.

This printer simulates media-size sensing capabilities. The printer informs the iOS printing system that it has US Letter paper loaded in the main tray, 5x7 inch paper loaded in the photo tray, and 4x6 inch paper in the alternate tray. iOS will therefore choose between these paper sizes regardless of what region the user has set. For photo printing, the user will be presented with the available sizes.

Simulated Laser

This printer simulates a black and white only PDF capable printer that supports 2-sided printing.

This printer simulates media-size sensing capabilities. The printer informs the iOS printing system that it has US Letter paper loaded in the main tray and 11x17 inch paper loaded in the alternate tray. iOS will therefore choose between these paper sizes regardless of what region the user has set. For photo printing, the user will be presented with the available sizes.
